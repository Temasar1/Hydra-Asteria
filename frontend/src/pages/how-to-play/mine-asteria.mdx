# Mine Asteria

Updates the `AsteriaUtxo` by transferring its value to the ship owner's address.  
The ship coordinates must match Asteria's coordinates `(0,0)` for this operation to be valid.  

## Hydra Example

You can use the following Hydra script to mine Asteria with your ship.  
The supporting files are available [here](https://github.com/MeshJS/asteria/tree/main/offchain).

<CodeBlock content={`import { mineAsteria } from "../src/mineAsteria.js";
import { hydraProvider, hydraWallet } from "../../utils.js";

async function main() {
    const ship_tx_hash =
        "594fb0c4baff88873d15d5973103efc667052deec3315ba4bc89aeaaf53dd407"; // ship utxo tx hash

    await hydraProvider.connect();

    const txHash = await mineAsteria(ship_tx_hash);

    console.log("Submitted Tx:", txHash);
}

main();
`} />
